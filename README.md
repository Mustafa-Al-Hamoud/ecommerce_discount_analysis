# Eniac Rabattstrategie Analysis [Python | Pandas]

## Project Overview

This project analyzes Eniac's discount strategy using Python, Pandas and data visualization techniques.

The objective is to clean and prepare the datasets, explore customer purchasing behavior, analyze the impact of discounts on sales, and visualize the main business insights.

---

## Key Findings

- Products with discounts generated higher sales activity.
- Average revenue differs between discounted and non-discounted products.
- Data cleaning was essential before performing the analysis.
- Visualizations made it easier to identify sales patterns and compare results.
